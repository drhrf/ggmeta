# Apply JAMA-style formatting

Modifies a forest plot to match JAMA (Journal of the American Medical
Association) style conventions.

## Usage

``` r
layout_jama(p)
```

## Arguments

- p:

  A ggplot object (typically from
  [`ggforest()`](https://drhrf.github.io/ggmeta/reference/ggforest.md)).

## Value

A ggplot object with JAMA-style formatting applied.
