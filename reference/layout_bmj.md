# Apply BMJ-style formatting

Modifies a forest plot to match BMJ (British Medical Journal) style
conventions.

## Usage

``` r
layout_bmj(p)
```

## Arguments

- p:

  A ggplot object (typically from
  [`ggforest()`](https://drhrf.github.io/ggmeta/reference/ggforest.md)).

## Value

A ggplot object with BMJ-style formatting applied.
